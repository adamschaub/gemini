# gemini roles

Individual roles should be cloned into this directory as follows:
```
git clone https://github.com/danehans/ansible-role-ansible-controller.git ~/gemini/ansible/roles/ansible-controller
git clone -b pxe_coreos https://github.com/danehans/ansible-role-httpd.git ~/gemini/ansible/roles/httpd
git clone -b pxe_coreos https://github.com/danehans/ansible-role-tftp.git ~/gemini/ansible/roles/tftp
git clone -b pxe_coreos https://github.com/danehans/ansible-dnsmasq.git ~/gemini/ansible/roles/dnsmasq
git clone -b pxe_coreos https://github.com/danehans/ansible-coreos-cloudinit.git ~/gemini/ansible/roles/coreos-cloudinit
```
